# Kali Linux Setup

## Obiettivo

Configurare una macchina virtuale Kali Linux per esercitarmi in cybersecurity in ambiente controllato.

## Ambiente utilizzato

| Voce | Dettaglio |
|---|---|
| Virtualizzazione | VirtualBox |
| Sistema guest | Kali Linux |
| Sistema host | Windows |
| RAM | 4 GB |
| CPU | 2 core |
| Disco virtuale | 40 GB |
| Rete | NAT |

## Attivita svolte

| Attivita | Scopo |
|---|---|
| Installazione Kali Linux | Creare la macchina principale del laboratorio |
| Aggiornamento sistema | Tenere aggiornati i pacchetti |
| Configurazione utente | Preparare l'ambiente di lavoro |
| Test connessione internet | Verificare accesso alla rete tramite NAT |

## Comandi utilizzati

| Comando | Scopo |
|---|---|
| sudo apt update | Aggiornare la lista dei pacchetti |
| sudo apt upgrade -y | Aggiornare i pacchetti installati |
| ip a | Controllare interfacce di rete e indirizzo IP |
| ping google.com | Testare la connettivita esterna |

## Competenze praticate

| Competenza | Descrizione |
|---|---|

| Linux setup | Preparazione di un ambiente Linux utilizzabile |
