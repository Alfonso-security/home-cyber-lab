# Kali Linux Setup

## Obiettivo

Configurare una macchina virtuale Kali Linux per esercitarmi in cybersecurity in ambiente controllato.

## Ambiente utilizzato

- VirtualBox
- Kali Linux
- Host system: Windows

## Configurazione VM

- RAM: 4 GB
- CPU: 2 core
- Disco virtuale: 40 GB
- Rete: NAT

## Attività svolte

- Installazione Kali Linux
- Aggiornamento sistema
- Configurazione utente
- Test connessione internet
