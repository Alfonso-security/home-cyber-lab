# 🛡️ Home Cyber Lab

Personal cybersecurity laboratory built using VirtualBox and Kali Linux.

This repository documents my practical learning journey in:
- Linux fundamentals
- Networking
- TCP/IP
- Packet analysis
- Wireshark
- Python networking
- Socket programming
- Basic cybersecurity concepts
- Network troubleshooting

The goal of this project is to develop practical cybersecurity skills through hands-on labs, packet analysis and Python-based networking tools.

---

# 🖥️ Lab Environment

| Component | Details |
|---|---|
| Host OS | Windows |
| Virtualization | VirtualBox |
| Guest OS | Kali Linux |
| RAM | 4 GB |
| CPU | 2 vCPU |
| Network Mode | NAT |
| Main Tools | Wireshark, tcpdump, Netcat, Python, Nmap |
