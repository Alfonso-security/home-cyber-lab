# Home Cyber Lab

Laboratorio personale creato per esercitarmi in cybersecurity in ambiente controllato.

## Strumenti utilizzati

- VirtualBox
- Kali Linux
- Ubuntu Server

## Obiettivi del laboratorio

- Imparare Linux
- Studiare networking
- Eseguire test in ambiente isolato
- Fare pratica con strumenti di sicurezza

## Attività future

- Configurazione rete virtuale
- Installazione macchine virtuali
- Utilizzo di Wireshark
- Utilizzo di Nmap
